列举一些我平常使用的软件，作为推荐同时也作为备忘。会保持更新。

效率工具

Snipaste

Windows | Linux || 去看看
非常好用的截图工具，除了截图，平常用的最多的是贴图功能。贴图在很多场景下都非常方便。

QuickLook

Windows | Linux || 去看看
QuickLook 可以增强 Window 资源管理器，在资源管理器中可以直接敲击空格键，预览文件。可以预览的文件格式很多，包括文档、图片、视频、压缩包等。

PowerToys

Windows | Linux || 去看看
PowerToys 是微软开源的一个工具集合，涵盖的种类非常多，包括取色器、屏幕分区、重命名等功能。

Notepad3

Windows | Linux || 去看看
最开始是使用 Notepad2 作为 Windows 记事本的替代品的，偶然发现了这个 Notepad3，相对 2 来说增加了一些功能，非常值得推荐。

编程相关

Visual Studio Code

Windows | Linux || 去看看 || 扩展列表
VSCode 绝对值得列在第一位，平常工作码代码以及日常码代码都离不开的优秀工具。

Windows Terminal

Windows | Linux || 去看看
同样微软出品的 Terminal ，比较好看，性能很好。同样推荐其内嵌的 Cascadia Code 字体。

ConEmu

Windows | Linux || 去看看 || 教程（待填坑）
Windows 平台上非常好用的多标签 terminal 工具，可以很方便地添加到在 Winodws 资源管理器右键菜单中。

sshfs

Windows | Linux || 去看看
sshfs 是一种基于 ssh 的文件传输协议，可以在不架设文件服务器的情况下，非常方便地访问到远程服务器上的文件，相对于 Windows 上用的较多的 samba 来说，要方便很多。上面给出的链接是 sshfs 在 WIndows 上的移植。

sshfs-win-manager

Windows | Linux || 去看看
sshfs GUI 工具，配合上面的一起食用，效果非常好。
