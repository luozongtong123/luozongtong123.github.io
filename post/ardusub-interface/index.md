# 树莓派 SSH 远程登陆  
这是最基本的交互方式，只要树莓派可以正常运行且连接到了同一个网络，我们就可以远程登陆到树莓派进行一些操作。ArduSub 中树莓派的账户密码如下：
``` json
{
  "user": "pi",
  "passward": "companion"
}
```

# 串口连接树莓派  
当树莓派甚至都无法连接到网络时，我们可以通过串口连接到树莓派，从而进行一些操作。关于树莓派你可能需要了解[这些](/tags/raspberry-pi/)。
